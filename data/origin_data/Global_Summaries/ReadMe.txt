Here you can find the global summaries. 

Note: Global summary is when there is no intended query. These are ideal to test 
conventional (as opposed to query-focused) video summarization frameworks.

Note II: You can find the paper and its supplementary material at:
https://www.aidean-sharghi.com/cvpr2017/

Within each folder, you can find 4 '.txt' files:
   1) 'user%i.txt': set of shots selected by the i^th user to be included in the summary.
   1) 'oracle.txt': this is the summary that is maximally aligned with 3 user summaries.
                    For more details, please refer to section B of the supplementary material.
